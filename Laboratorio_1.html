¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Es un método que sirve para obtener información del servidor web. No cambia el estado de los datos.
  • Head: También sirve para obtener información del servidor, sin embargo, este sólo pide el encabezado de esta forma se puede saber si está disponible antes de hacer la solicitudcon el GET.
  • POST: Se usa para enviar información al servidor.
  • PUT: Sirve para alterar la información del cuerpo.
  • PATCH: Modifica parcialmente la página.
  • DELETE: Borra una parte específica de la página.
